## 🎮 游戏简介

一个轻策略卡牌式股票模拟网页原型，核心循环是：  
**开盘 → 增持 → 交付 → 强化**

## ✨ 核心特色

- 12 轮目标推进，逐轮冲刺交易额
- 多资金池协同（普通池 / 助涨池 / 保险池）
- 交易卡与强化卡双线成长
- 开盘结果即时反馈（上涨红、下跌绿）

## 🚀 本地运行

直接用浏览器打开 `index.html` 即可。

## 📁 结构

- `index.html`：主入口
- `scripts/`：逻辑与配置
- `styles/`：界面样式
- `assets/`：美术与音频
- `docs/`：策划与说明
