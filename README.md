## 项目简介

一个轻策略卡牌式股票模拟网页原型，围绕「开盘 -> 增持 -> 交付 -> 强化」形成单局循环。

## 核心玩法

- 12 轮关卡目标推进，逐轮完成交易额要求
- 多资金池协同（普通池 / 助涨池 / 保险池）
- 回合内交易卡决策与回合后强化成长并行
- 开盘结果即时可视化反馈（上涨/下跌）

## 目录结构

- `index.html`：主入口页面
- `scripts/`：核心逻辑与配置
- `styles/`：样式文件
- `assets/`：美术与音频资源
- `docs/`：策划文档与设计说明
